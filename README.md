## Developing

Clone the repository:

```bash
git clone https://github.com/flurium/spentoday-frontend.git
```

Get inside of folder:

```bash
cd spentoday-frontend
```

Use dev branch:

```bash
git checkout dev
```

From install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```
